# はじめてのプログラミング

## やってみよう

ターミナルで以下のコマンドを打ってみましょう。

### 1. 文字を表示する

```bash
echo "Hello, World!"
```

> `echo` は「画面に文字を表示する」命令です。

---

### 2. スクリプトを実行する

ファイルにまとめた命令をまとめて実行できます。

```bash
bash hello.sh
```

実行すると、こんなふうに表示されます：

```
Hello, World!
はじめてのプログラミングへようこそ！

いまの日時: Thu Feb 12 07:00:00 JST 2026
```

---

## プログラミングって何？

コンピュータへの「命令書き」です。

| 言葉       | 意味                         |
|------------|------------------------------|
| `echo`     | 画面に文字を表示する         |
| `#`        | コメント（コンピュータは無視）|
| `$(date)`  | 日時を取得する               |

命令を並べたファイルのことを **スクリプト** と呼びます。
